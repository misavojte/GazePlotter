# Participant Groups

Participant groups let you organize participants into sets for comparison across plots and metrics. Each plot can target *All participants* or one group.

## Opening the editor

1. Select a plot to open its [Settings Pane](/docs/visualizations/#configuration-and-settings-pane).
2. In the **Participant group** section, click **Edit groups…**.

## Managing groups

- **Add group** — The button at the bottom creates a group named `Group 1`, `Group 2`, … Edit the name inline in the group header.
- **Reorder** — Drag a group by its grip handle. Order sets how groups appear in plots.
- **Duplicate** — The copy icon clones a group with its members.
- **Delete** — The trash icon removes a group. Non-empty groups ask for confirmation first.

## Assigning participants

Expand a group (click its participant count) to assign members:

- Check or uncheck a participant to add or remove them.
- **Select all** / **Deselect all** apply to the whole list.
- **Search participants** filters the list by displayed name. While a filter is active, the buttons become **Select visible** / **Deselect visible**, acting only on matches. Selections made under one filter persist when you clear it, so you can build a group across several searches.

## Saving

Click **Apply** to save the groups, or **Cancel** to discard all changes.
